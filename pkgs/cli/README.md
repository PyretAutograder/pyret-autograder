# pyret-autograder-cli

> [!WARNING]
> The cli package currently only works for pawtograder. If you would like to run the gradescope
> grader locally, use docker. See [Gradescope's docs][1] for more information.

[1]: https://gradescope-autograders.readthedocs.io/en/latest/manual_docker/#running-autograder-images-locally

A command-line interface for running pyret-autograders locally.

## Usage

Npm will automatically download the latest version of this package if you run

```
npx pyret-autograder-cli --help
```

The package also exposes an executable under the name `pyret-autograder`
